<Steps>
  <Step title="Get Quote">
    Let's get a quote to trade 0.001 <img src="/images/assets/bitcoin.svg" width="20" style={{display: 'inline'}} /> on **Bitcoin Testnet4** to <img src="/images/assets/wbtc.svg" width="20" style={{display: 'inline'}} /> on **Ethereum Sepolia**.
    ```bash wrap
    curl -X 'GET' \
     'https://testnet.api.garden.finance/quote/?order_pair=bitcoin_testnet:primary::ethereum_sepolia:0x88e1032252c712681A3659986F452E910D2134D8&amount=100000&exact_out=false' \
      -H 'accept: application/json'
    ```
    ```json Response wrap
    {
        "status": "Ok",

                "btyrea8d": "99700"
    ```
